▸ send(header, body, opts)

Make a call to Flowee This should be used with messages that expect a reply

send(header: object, body: object, opts: object): Promise<Message>
Parameters
header (object) An object containing the header (should be enums)
body (object) An object containing the body (should be enums)
opts (object = {}) An object containing options for the request
Returns
Promise<Message>: Resolves with the reply or rejects if error
Example
let res = await flowee.send({ [Tags.ServiceId]: 4, [Tags.MessageId]: 0 }, {
  2: address,
  6: amount
});

▸ addHandler(filter, callback)

Add listener that will process replies that match filter

addHandler(filter: any, callback: any)
Parameters
filter (any) A object that contains headers
callback (any) Callback when matches
Example
flowee.addHandler({ [Tags.ServiceId]: 0, [Tags.MessageId]: 2}, (msg) => {
  msg.extraInformation = "You can transform the message payload here";
  return msg;
});
